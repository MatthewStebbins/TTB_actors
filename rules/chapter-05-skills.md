# Chapter 5: Skills

Skills represent the education, training, and learned abilities that a character has acquired over the course of her life. There is no limit to the number of Skills a character can know, and a character's Skills increase fairly frequently.

Skills are grouped into rough **Types** — descriptors that help define and organize similar Skills into loose groups. The rules often reference Skills by their Type (e.g., "all Magical Skills" or "any Crafting Skill").

**Skill Types:**
- Academic
- Close Combat
- Crafting
- Expertise
- Magical
- Ranged Combat
- Social
- Training

At the end of each session, each character will typically advance in at least one Skill (see Character Advancement, pg. 318).

## Associated Aspects

Each Skill has an **Associated Aspect** — the Aspect most commonly added to Skill ranks to determine the character's **Acting Value** (pg. 284).

The Fatemaster may rule that a different Aspect is more relevant when a Skill is used in an unusual manner. For example, the Shotgun Skill normally uses Grace, but assembling a dismantled shotgun under time pressure might call for a Shotgun Challenge using Speed instead.

## Skill Reference Tables

### Academic Skills

| Skill | Associated Aspect | Description |
|---|---|---|
| Bureaucracy | Cunning | The inner workings of legal and government matters. |
| Engineering | Intellect | Proper mechanical and structural design. |
| History | Intellect | The lore of human existence. |
| Literacy | Intellect | Reading and deciphering text, learning new languages. |
| Mathematics | Intellect | Arithmetic, Physics, and other numerical disciplines. |
| Music | Charm | The ability to compose and play music. |

### Close Combat Skills

| Skill | Associated Aspect | Description |
|---|---|---|
| Flexible | Grace | Chains, Lassos, Whips, and any other such weapons. |
| Grappling | Speed | Wrestling and holding an opponent. |
| Heavy Melee | Might | Large and cumbersome weapons that require two hands. |
| Martial Arts | Speed | Quick strikes to vulnerable areas made with hands and feet. |
| Melee | Might | One handed weapons that bludgeon, slash, or pierce. |
| Pneumatic | Might | Weapons that include mechanical parts. |
| Pugilism | Might | Powerful strikes usually made with closed fists. |

### Crafting Skills

| Skill | Associated Aspect | Description |
|---|---|---|
| Alchemistry | Intellect | Mixing and properly using chemicals. |
| Art | Cunning | Producing and understanding artistic works. |
| Artefacting | Intellect | Building constructs and machinery. |
| Blacksmithing | Intellect | Forging metal objects such as swords and horseshoes. |
| Culinary | Charm | Creating appetizing food and drink. |
| Explosives | Intellect | Creating and using bombs, dynamite, and other explosives. |
| Homesteading | Tenacity | Making and maintaining the materials of everyday life. |
| Printing | Intellect | Creating books and other written objects. |
| Stitching | Tenacity | Sewing and stitching of both fabric and corpses. |

### Expertise Skills

| Skill | Associated Aspect | Description |
|---|---|---|
| Doctor | Intellect | Healing the sick and injured. |
| Forgery | Cunning | Creating false documents and writings. |
| Gambling | Cunning | Properly playing games of chance and skill. |
| Husbandry | Charm | Caring for livestock and riding/driving transport animals. |
| Lockpicking | Grace | Opening locks and other complex devices. |
| Notice | Cunning | Spotting things that are out of place or hidden. |
| Track | Cunning | Following a target through an environment. |
| Wilderness | Cunning | Proper survival and safety in unsettled areas. |

### Magical Skills

| Skill | Associated Aspect | Description |
|---|---|---|
| Counter-Spelling | Tenacity | Stopping magical effects and spells. |
| Enchanting | Charm or Cunning | Imbuing an object or target with magical qualities. |
| Necromancy | Charm or Tenacity | Control over another's mind, body, or spirit. |
| Prestidigitation | Cunning or Intellect | Manipulation of space, time, and the senses. |
| Sorcery | Intellect or Tenacity | Creation of elemental effects. |

> **Note**: Magical Skills with two Associated Aspects use whichever is higher (or the character's choice when equal), unless the Fatemaster rules otherwise.

| Pistol | Grace | One handed firearms like derringers and dueling pistols. |
| Shotgun | Grace | Shotgun weapons of both the scatter and slug variety. |

## Quick Reference: All Skills by Aspect

| Aspect | Skills |
|---|---|
| Might | Heavy Melee, Melee, Pneumatic, Pugilism, Athletics, Heavy Guns |
| Grace | Flexible, Lockpicking, Acrobatics, Archery, Pistol, Shotgun, Thrown Weapons |
| Speed | Grappling, Martial Arts, Evade, Pick Pocket |
| Resilience | Carouse, Toughness |
| Charm | Music, Culinary, Husbandry, Bewitch, Leadership, Enchanting\*, Necromancy\* |
| Cunning | Bureaucracy, Art, Forgery, Gambling, Notice, Track, Wilderness, Deceive, Scrutiny, Stealth, Enchanting\*, Prestidigitation\* |
| Intellect | Engineering, History, Literacy, Mathematics, Alchemistry, Artefacting, Blacksmithing, Explosives, Printing, Doctor, Long Arms, Convince, Sorcery\*, Prestidigitation\* |
| Tenacity | Homesteading, Stitching, Barter, Intimidate, Centering, Counter-Spelling, Sorcery\*, Necromancy\* |

> \* Magical Skills with dual Associated Aspects appear under both; use the higher Aspect value when calculating Acting Value.
